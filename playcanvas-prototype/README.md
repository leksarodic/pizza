# PlayCanvas Prototype

This folder is a fresh PlayCanvas scaffold created on the `codex/playcanvas-scaffold` branch so we can evaluate a browser-first engine path for Pizza Afterglow without replacing the current Three.js version.

## Run

1. Install dependencies:

```bash
cd playcanvas-prototype
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Current scope

- PlayCanvas app bootstrap
- Basic evening scene
- Placeholder roads, buildings, pizza shop, and car
- HTML overlay for migration notes
